\begin{itemize}
\item Tanie transakcje

\item Częste transfery tokenów
\item Duża ilość 'wielorybów'- Obecność użytkowników posiadających znaczące ilości aktywów

\item Możliwość rozwinięcia projektu w przyszłości
\end{itemize}
